# **Barbearia Na Navalha** 

**Aplicação de gestão para barbearias**, desenvolvida com **ASP.NET MVC**, focada em otimizar o tempo dos barbeiros e melhorar a experiência tanto do profissional quanto do cliente.

## **Descrição**

Este projeto é uma aplicação desenvolvida em **ASP.NET MVC** com o objetivo de otimizar a gestão de tempo e os agendamentos para barbearias. A aplicação facilita o processo de agendamento de horários, permitindo que os barbeiros gerenciem seus compromissos de forma rápida e eficiente. 

Além disso, a interface intuitiva e o design moderno garantem uma experiência agradável tanto para os profissionais quanto para os clientes. A aplicação foi construída com a intenção de ajudar a otimizar o tempo dos barbeiros, aumentar a produtividade e melhorar a comunicação com os clientes, proporcionando um ambiente mais organizado e ágil.

## **Tecnologias Utilizadas**

- **ASP.NET MVC**: Framework utilizado para a construção da aplicação web.
- **C#**: Linguagem de programação principal para o desenvolvimento da lógica de backend.
- **SQL Workbench**: Banco de dados utilizado para armazenar informações de agendamentos, clientes e barbeiros.
- **Entity Framework**: ORM utilizado para facilitar a interação com o banco de dados e realizar consultas de maneira eficiente.
- **Razor**: Motor de templates para renderização de páginas dinâmicas.
- **Bootstrap**: Framework CSS utilizado para garantir um design moderno, responsivo e agradável ao usuário.
- **ASP.NET Identity**: Sistema de autenticação e gerenciamento de usuários (com suporte para administração de permissões e senhas seguras).

## **Funcionalidades**

- **Agendamento de Horários**: Permite que os clientes agendem horários com os barbeiros diretamente pela plataforma, evitando filas e melhorando o controle de tempo.
  
- **Painel de Administração**: Funcionalidade para o administrador gerenciar barbeiros, agendamentos, horários e clientes, garantindo a flexibilidade necessária para administrar a barbearia de maneira eficiente.

- **Gerenciamento de Usuários**: Possibilidade de os usuários se cadastrarem, fazerem login e gerenciarem seus agendamentos, com suporte a diferentes níveis de permissão para barbeiros e administradores.

- **Design Responsivo e Intuitivo**: A aplicação é projetada para ser visualmente atraente e fácil de usar, tanto em dispositivos móveis quanto em desktop.

- **Sistema de Notificações**: O cliente recebe notificações de confirmação de agendamento, enquanto os barbeiros recebem lembretes sobre horários agendados.
